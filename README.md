# Nimbus Weather Dashboard

A sleek, modern weather dashboard built with **HTML, CSS, and JavaScript**, inspired by minimal UI and responsive design principles.  
Nimbus provides real-time weather data, forecasts, air quality insights, and sun cycle visualization — all powered by the **OpenWeatherMap API**.

![Nimbus Weather Dashboard Preview](./preview.png)

---

## Overview

Nimbus Weather delivers weather intelligence in a visually rich, user-friendly interface.  
Designed with aesthetic precision and modular architecture, it provides detailed daily metrics like:

-  Current temperature, humidity, and visibility  
-  Animated sunrise–sunset arc with live sun position  
-  UV index with health-level indicator  
-  Real-time air quality index and pollutant details  
-  Temperature trend throughout the day  
-  5-day weather forecast summary  

Everything updates automatically based on the searched or current location.

---

##  Features

| Feature | Description |
|----------|-------------|
| **Dynamic Location Search** | Enter any city name to fetch live weather and forecast data. |
| **Current Weather Card** | Displays real-time conditions like temperature, humidity, pressure, and visibility. |
| **Air Quality Card** | Shows AQI level, pollutant type, and wind details with color-coded indicators. |
| **Sun Cycle Visualization** | A clean SVG-based arc shows sunrise and sunset times with a moving sun icon. |
| **UV Index Indicator** | Merged with the sun card to provide UV risk level and category (“Low”, “Moderate”, “High”). |
| **Temperature Timeline** | Illustrates temperature changes across morning, afternoon, evening, and night. |
| **Forecast Summary** | Displays a short-term weather prediction with date, icons, and temperature range. |
| **Lucide Icons** | Consistent, vector-based iconography for all conditions and UI components. |
| **Responsive Layout** | Fully adaptive grid design — optimized for desktops, tablets, and mobile screens. |

---

##  Tech Stack

- **HTML5** — Semantic structure for modular cards and layout  
- **CSS3 (Flex + Grid)** — Modern responsive UI, soft glassmorphism styling  
- **JavaScript (ES6+)** — Dynamic data fetching and DOM updates  
- **OpenWeatherMap API** — Real-time weather and forecast data  
- **Lucide Icons** — Lightweight open-source icon library
